SECTION 0

개요

Poly AI Playground 플랫폼의 교육용 게임 콘텐츠는 AI를 활용하여 생성 및 제작되었습니다. 사용자에게 이를 투명하게 안내하기 위해 AI 생성 콘텐츠 표기를 플랫폼 전반에 추가합니다.

본 가이드는 개발자가 일관된 방식으로 AI 표기를 구현할 수 있도록 표기 문구, 디자인 시안, 코드 스니펫, 에셋을 제공합니다. 각 항목은 바로 복사하여 사용할 수 있습니다.

SECTION 4

콘텐츠 인트로 화면

각 게임/액티비티에 진입한 후 표시되는 인트로 화면 하단에 AI 생성 고지 배너를 표시합니다. 게임 테마에 맞는 다크/라이트 변형을 사용합니다.

디자인 목업 - 다크 테마 (Chef's Chat Kitchen)
이 콘텐츠는 AI에 의해 생성되었습니다.
디자인 목업 - 라이트 테마 (Animal Explorer)
이 콘텐츠는 AI에 의해 생성되었습니다.
💡 배너는 인트로 화면의 우측 하단에 배치합니다. 게임 플레이에 방해가 되지 않도록 반투명 처리합니다.

React JSX 컴포넌트

JSX 
/**
 * AI 생성 콘텐츠 고지 배너
 * @param {'dark' | 'light'} theme - 게임 테마에 맞춰 설정
 */
const AiGeneratedBanner = ({ theme = 'dark' }) => {
  const isDark = theme === 'dark';

  return (
    <div
      className={`flex items-center gap-2 px-5 py-2.5 rounded-xl
        text-xs font-medium
        ${isDark
          ? 'bg-white/[0.08] text-white/70 border border-white/10'
          : 'bg-black/[0.06] text-black/50 border border-black/[0.08]'
        }`}
    >
      <svg
        width="16" height="16"
        viewBox="0 0 24 24"
        fill="none"
        stroke="currentColor"
        strokeWidth="2"
        strokeLinecap="round"
        strokeLinejoin="round"
      >
        <path d="M12 2l2.4 7.4H22l-6.2 4.5L18.2 22 12 17.3 5.8 22l2.4-8.1L2 9.4h7.6z" />
      </svg>
      이 콘텐츠는 AI에 의해 생성되었습니다.
    </div>
  );
};

export default AiGeneratedBanner;

SECTION 5

대화 기능 AI 표시

Chef's Chat Kitchen 등 AI와 실시간 대화하는 게임에서는 대화창 내에 AI 생성 사실을 표시해야 합니다. 매번 반복 표시할 필요 없이 초기 1회 안내 또는 지속 아이콘 표출로 의무를 충족합니다.

💡 인공지능 기본법 제31조 제2항 — 대화 기반 서비스의 경우 초기 화면 표시 또는 로고 지속 표출 방식으로 AI 생성 사실 표시 가능. 개별 대화마다 반복 표시 불필요.

디자인 목업

디자인 목업 (AI 대화 표기)
AI 대화
디자인 목업 2 (AI 대화 표기)
AI 대화

구현 방법 (2가지)

① 대화창 헤더에 AI 아이콘 + 레이블 지속 표출 — 추천. 사용자가 항상 인지 가능하며 반복 표시 불필요.
② 대화 시작 시 첫 메시지로 AI 안내 문구 1회 출력 — 대화창 진입 시 자동으로 "이 대화는 AI에 의해 생성됩니다." 메시지 표시.

React JSX 코드

JSX 
/* 대화창 헤더 - AI 표시 레이블 */
<div className="flex items-center justify-between px-4 py-3 bg-slate-900 rounded-t-xl">
  <span className="text-white font-bold text-sm">Chef's Chat Kitchen</span>
  <span className="flex items-center gap-1.5 px-2.5 py-1 rounded-full
    bg-white/[0.08] text-white/65 text-[10px] font-medium
    border border-white/[0.12]">
    <AiSparkleIcon size={11} />
    AI 대화
  </span>
</div>

/* 대화창 하단 안내 문구 */
<div className="flex items-center justify-center gap-1.5
  text-white/40 text-[10px] py-2">
  <AiSparkleIcon size={11} />
  대화 내용은 AI에 의해 생성됩니다.
</div>
SECTION 6

다운로드 시 파일 표시

학습 결과물, 리포트 등 콘텐츠를 다운로드·공유하는 기능을 제공하는 경우, 파일 내부 또는 다운로드 시점에 AI 생성 사실을 표시해야 합니다.

💡 인공지능 기본법 제31조 제2항 — 서비스 내 UI에 표시하였더라도, 다운로드·공유 기능 제공 시에는 결과물 자체에 별도로 AI 생성 사실을 표시해야 함.

표시 방법 (유형별)

파일 유형표시 위치방법
PDF / 문서 문서 머리말 또는 첫 페이지 하단 AI 생성 안내 문구 삽입
이미지 파일 메타데이터 또는 이미지 하단 워터마크 또는 EXIF 메타데이터
모든 파일 다운로드 시점 토스트/팝업으로 1회 안내

다운로드 토스트 안내 (React JSX)

JSX 
/**
 * 파일 다운로드 시 AI 생성 사실 토스트 안내
 * 다운로드 함수 내에서 호출
 */
const handleDownload = async (fileUrl, fileName) => {
  // 1. 다운로드 실행
  const link = document.createElement('a');
  link.href = fileUrl;
  link.download = fileName;
  link.click();

  // 2. AI 생성 안내 토스트 표시 (1회)
  toast(<div className="flex items-center gap-2">
    <AiSparkleIcon size={14} />
    <span>이 파일은 AI가 생성한 콘텐츠를 포함하고 있습니다.</span>
  </div>, {
    duration: 4000,
    icon: '🤖',
  });
};

⚠️ 이 항목은 개발 작업이 아닌 운영/법무팀 협의 사항입니다. 아래 문구를 참고하여 이용약관 담당자에게 전달하세요.

📋 인공지능의 발전과 신뢰 기반 조성 등에 관한 기본법 제31조 제1항 — 고영향 AI 또는 생성형 AI를 이용한 제품·서비스 제공 시, 해당 제품·서비스가 AI에 기반하여 운용된다는 사실을 이용자에게 사전에 고지해야 함. 이용약관·계약서에 기재하는 방식으로 의무 충족 가능.
